# IniParser

Simple case sensitive ini file parser, .NET Standard 2.0.

`Install-Package WinSoftIniParser`

## Support

Values.

`Property name = Value`

Values with comment.

`Property name = Value; This is a comment.`

Sections.

`[This is a new section]`

Sections with comment.

`[This is a new section]; This is a comment.`

Comments.

`; This is a comment.`
